# Mercado Privado Chile

Plataforma web de compra y venta privada para usuarios en Chile.

## Estructura

```
mercadoprivadochile/
└── index.html      # Página principal
```

## Características

- Diseño **mobile-first** con navegación inferior en dispositivos móviles
- Navegación superior en desktop (≥ 768px)
- Iconos con [Font Awesome 6.5](https://fontawesome.com/)
- Layout responsive con CSS Grid

## Uso

Abrir `index.html` directamente en el navegador. No requiere servidor ni dependencias instaladas.

## Breakpoints

| Breakpoint | Layout |
|---|---|
| `< 768px` | Nav fija en la parte inferior, cards en 1 columna |
| `≥ 768px` | Nav en el header, cards en 3 columnas |

## Tecnologías

- HTML5
- CSS3
- Font Awesome 6.5 (CDN)
